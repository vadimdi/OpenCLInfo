OpenCLInfo
==========

OpenCLInfo: a tool for generation of exhaustive report for all available OpenCL computing devices

If it is possible, please send your report generating on your hardware to contact@hgpu.org for publishing.
(OpenCLInfo does not provide any communication operations, please send it manually via email).

All available data is collecting on hgpu.org site (http://hgpu.org/?cat=32).
